# JCC Restaurant Website

A professional restaurant website with a modern design, featuring home, menu, reservations, and contact information.

## 📁 Project Structure

```
JCC-Restaurant/
├── index.html          # Main homepage
├── css/
│   └── styles.css      # All styling and responsive design
├── js/
│   └── script.js       # Interactive features and animations
└── images/             # Folder for restaurant images
```

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Navigation Bar** - Sticky navigation with smooth scrolling
- **Hero Section** - Eye-catching banner with call-to-action
- **About Section** - Restaurant information and highlights
- **Menu Section** - Organized menu with appetizers, mains, and desserts
- **Reservation System** - Form for customers to book tables
- **Contact Information** - Location, phone, hours, and email
- **Interactive Elements** - Smooth animations and hover effects
- **Mobile Menu** - Hamburger menu for mobile devices

## 🚀 How to Use

1. **Open the website**: Simply open `index.html` in your web browser
2. **Navigate sections**: Use the navigation menu to scroll to different sections
3. **Make a reservation**: Fill out the reservation form to book a table
4. **Contact info**: Find restaurant details in the contact section

## 🎨 Customization

### Colors
Edit the CSS variables in `css/styles.css` (lines 11-17):
```css
:root {
    --primary-color: #8B4513;      /* Brown */
    --secondary-color: #D2B48C;    /* Tan */
    --accent-color: #C9102E;       /* Red */
    --dark-color: #1a1a1a;         /* Dark */
    --light-color: #f8f8f8;        /* Light */
    --text-color: #333;            /* Text */
}
```

### Restaurant Information
Edit `index.html` to update:
- Restaurant name and details
- Menu items and prices
- Contact information (phone, email, address)
- Hours of operation

### Images
Add your restaurant images to the `images/` folder and update image paths in `index.html`

## 📱 Responsive Breakpoints

- **Desktop**: Full layout with side-by-side content
- **Tablet**: Adjusted grid layouts (max-width: 768px)
- **Mobile**: Single column layout with stacked content (max-width: 480px)

## 💡 JavaScript Features

- Smooth scrolling navigation
- Hamburger menu for mobile

- Scroll animations for menu items
